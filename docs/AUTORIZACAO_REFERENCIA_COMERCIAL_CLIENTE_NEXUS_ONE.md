# Autorizacao de Referencia Comercial do Cliente - Nexus One

Data de referencia: 10/05/2026

Use este documento quando um cliente estiver saudavel apos implantacao e houver interesse em usar o caso como referencia comercial, depoimento, logo, segmento, resultado ou prova de valor.

## Regras

- Nunca usar nome, logo, depoimento, numero, print ou resultado do cliente sem autorizacao formal.
- Nunca expor dados pessoais, dados financeiros, documentos fiscais, nomes de clientes finais, prints com informacao sensivel ou segredo operacional.
- Preferir caso anonimizado quando a autorizacao nao for ampla.
- A autorizacao deve indicar exatamente o que pode ser usado.
- O cliente pode revogar ou limitar a autorizacao conforme contrato/acordo.
